How it works
Enter your GitHub credentials below. The tracker will save all your data to a tracker-data.json file in your GitHub repo. When you hit ⬆ Sync to GitHub (or enable auto-sync), your live GitHub Pages tracker always stays up to date — accessible from any device.
GitHub Configuration
Create one at github.com/settings/tokens — enable the repo scope. It stays in your browser only.
